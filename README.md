# Alpha Token Volume Board

Binance Alpha 代币 24H 交易量看板：实时展示 Alpha 代币价格、24H 交易量、FDV、当天/前一天限价单成交额，支持交易赛累计成交额统计、区间查询和小时交易量图。

## 功能

- **排行榜**：价格 / 24H 交易量 / 24H 变化 / FDV / 当天与前一天限价单成交额（以每日 UTC+8 早 8 点为边界），支持点击表头排序
- **标记**：30 天内上新（🆕 剩 X 天）、币安合约（合约）、交易量加成活动（🔥 剩 X 天）、交易赛（🏆 含倒计时 / 奖池 / 人均奖励悬停提示）
- **筛选**：仅看 30 天内上新 / 有合约 / 代币化证券 / 交易赛，搜索框按名称/符号过滤；筛选与排序状态自动保存
- **交易赛**：累计成交额实时统计（支持代币化证券聚合赛、Aster 合约赛合成条目），总交易量点击复制
- **区间查询**：任意时间段限价单成交额 + 笔数 + 单笔均额，自动选择最细 K 线粒度；查询记录本地保存
- **多代币对比**：区间查询输入框用逗号/空格分隔多个代币（最多 10 个），按成交额降序对比
- **小时交易量**：近 N 小时（1~72）每小时成交额柱状图
- **迷你走势图**：当天限价单成交额列内嵌逐小时 sparkline（复用已有请求，零额外开销）
- **URL 分享**：筛选/排序/搜索状态同步到 URL hash，复制链接即可分享当前视图
- **其它**：收藏置顶、中英双语、深浅色主题、CSV 导出、移动端适配

## 运行 / 部署

纯静态单页应用，无构建步骤：

```bash
# 本地预览（任意静态服务器均可）
npx serve .
# 或
python3 -m http.server 8000
```

部署到 Vercel：直接导入仓库即可（已内置 `/_vercel/insights` 统计脚本）。

> 注意：直接以 `file://` 打开时 `config/activities.json` 无法加载，会自动回退到 `index.html` 内置的默认活动配置。

## 活动配置维护（config/activities.json）

**上/下交易赛、加成活动只需改这个 JSON，无需改代码。** 加载失败时回退到 `index.html` 中 `DEFAULT_ACTIVITIES` 的内置默认值（建议两处同步更新，内置值作为兜底）。

### competitions（交易赛）字段

| 字段 | 必填 | 说明 |
|---|---|---|
| `symbol` | ✅ | 代币符号（与 Alpha 列表的 ticker 一致；合成条目可用任意名称） |
| `start` / `end` | ✅ | 起止时间，ISO 字符串，如 `"2026-06-02T21:00:00+08:00"` |
| `topN` | ✅ | 参与资格名次 |
| `pool` | ✅ | 奖池文案，如 `"588,000 ZEST"` |
| `perWinner` | ✅ | 人均奖励文案，如 `"240 ZEST"`（用于折算 USD） |
| `rewardSymbol` | | 奖励代币符号（奖励非本币时用于查价折算 USD） |
| `syntheticToken` | | `true` 时注入独立合成行（列表中不存在该 ticker 时） |
| `aggregateSec` | | `true` 时成交额 = 全部代币化证券成交额之和（代币化证券总交易大赛） |
| `icon` | | 合成行的 Logo URL |
| `api` | | `"aster"` 时数据走 Aster 合约 API（配合 `asterSymbol`，如 `"XXXUSDT"`） |
| `isFutures` / `baseSymbol` | | Aster 合约赛：显示合约标记 / 复用基础代币的 Logo 与 FDV |
| `referralUrl` / `referralKey` | | 行内返佣链接及其 i18n 文案键 |

### volumeBoosts（交易量加成活动）

`合约地址(小写) -> 截止时间 ISO 字符串`，登记后该代币显示 🔥 剩 X 天 倒计时，到期自动消失。

## 技术说明

- 单文件 `index.html`（HTML/CSS/JS），数据来源：Binance Alpha 公共接口（代币列表 + K 线）、币安合约 `exchangeInfo`、Aster 合约 API（仅合约赛合成条目）
- 成交额请求并发数 15，失败时温和重试一次；**请求失败 ≠ 0**：失败显示 ⚠ 或沿用上一轮旧值（标记 stale）
- 前一天成交额按天缓存（localStorage，跨天自动失效），自动刷新（5 分钟）只重新拉取当天数据
- 文案为 `data-i18n` 驱动：新增文案给元素加 `data-i18n` / `data-i18n-ph` / `data-i18n-title` 属性并补 `I18N` 键即可
